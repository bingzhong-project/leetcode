# Convert Sorted Array to Binary Search Tree

[问题描述](https://leetcode.com/problems/convert-sorted-array-to-binary-search-tree/)

## 知识点

深度优先搜索

## 解题思路

根据二叉平衡查找树的特性，左子节点必定比根节点小，右子节点必定比根节点大。除此之外，还需要保证树的平衡。  
而给定的数组是有序的，每次都找出数组的中心点作为节点，左子树由左子数组构造，右子树由右子数组构造。由于作为分界分离数组的是中心元素，所以左子数组和右子数组的元素个数相差不会大于 1 ，这样就可以保证左子树的值必定小于父节点，右子树的值必定大于父节点，而且树的高度不会相差大于 1 。
